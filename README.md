# HT26 Interactive Course Planner

This is a local React/Vite app for viewing course schedules, uploading cohort and timetable files, and spotting clashes across the term.

## Run locally

Prerequisite: Node.js

1. Install dependencies with `npm install`
2. Start the development server with `npm run dev`
3. Open the local Vite URL shown in the terminal

## Notes

- No API key is required for the current version of this app.
- Uploaded data stays in the browser until you export or replace it.
